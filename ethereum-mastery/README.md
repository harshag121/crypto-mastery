## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd crypto-mastery/ethereum-mastery
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment:**
   ```bash
   cp .env.example .env
   # Add your Ethereum RPC URLs and private keys
   ```

4. **Start with Module 1:**
   ```bash
   cd 1-EthereumBasics
   node main.js
   ```
